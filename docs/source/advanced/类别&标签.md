
# 类别&标签

在配置文件`_config.yml`中有属性

    # Category & Tag
    default_category: uncategorized
    category_map:
    tag_map:

* 属性`default_category`表示默认类别
* 属性`category_map`可用做类别映射，比如将中文类别映射成英文，将多个类别映射到同一个
* 属性`tag_map`的用法和`category_map`类似，将标签进行映射

示例如下：

    category_map:
      随笔: essay
      测试: test
    tag_map:
      数学: math
      编程: programming
