

# 布局

`NexT`提供了`4`种布局(或称为主题、`scheme`)

    Muse 主题
    Mist 主题
    Pisces 主题
    Gemini 主题

其预览界面可在[hexo-theme-next](https://github.com/theme-next/hexo-theme-next/blob/master/docs/zh-CN/README.md#%E5%8D%B3%E6%97%B6%E9%A2%84%E8%A7%88)中点击相应网站查看

在`NexT _config.yml`文件中修改

    # Schemes
    # scheme: Muse
    # scheme: Mist
    # scheme: Pisces
    scheme: Gemini
